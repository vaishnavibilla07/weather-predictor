# Weather Predictor

## Description

The Weather Predictor website is a simple and intuitive tool that allows users to check the current weather conditions in any city around the world. By entering the city name, users can receive instant weather updates, including temperature, sky conditions, and wind speed. The website is powered by the OpenWeatherMap API, ensuring that the data is accurate and up-to-date.

## Features

- **City-based Weather Search:** Enter any city name to get the latest weather data.
- **Real-time Data:** Fetches live data from the OpenWeatherMap API.
- **Weather Details:** Displays the current temperature, sky conditions, and wind speed.
- **Responsive Design:** Optimized for various devices, providing a user-friendly experience.

## Technologies

- **HTML5:** Structure and layout of the website.
- **CSS3:** Styling and responsive design.
- **JavaScript:** Handles API requests and updates the UI dynamically.
- **OpenWeatherMap API:** Provides weather data.
